### 🛠 Setting-up

```
git clone https://github.com/FemboyClub/Netflix-Checker
cd Netflix-Checker
pip install -r requirements.txt
python main.py
```
---------------------------------------

### 🏆 Features 
- Ultra fast hits.
- Custom algorithm to minimise cpu usage. 
- Proxy support (HTTPS, SOCKS4, SOCKS5).
- High CPM.
- Save hits to file.
